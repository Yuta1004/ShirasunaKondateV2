# 白砂寮献立アプリ ver2
白砂寮献立アプリ ver2です  
UIや機能を一新し、Kotlinでコーディングを行います 

- 実装予定機能
  - メイン
    - 献立表示(実装済)
    - DB連携
    - 前日・明日表示ボタン(実装済)
  - 検索
    - 献立検索(実装済)
  - 日時指定
    - 献立を日時指定で表示(実装済)
  - 通知
    - アプリアップデート通知
    - 寮からのお知らせ一覧表示
  - その他
    - 設定

- ver1からの変更点(予定)
  - アプリ起動時にサーバと通信を行い、アプリアップデートがあればポップアップで通知
  - DBと連携し、美味しさ等を記録できるように
  - お知らせ画面を追加
  - UI変更
  - 献立ランダム機能廃止
